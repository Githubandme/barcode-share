# 扫码分享 (Scan & Share)

一款简洁、高效的安卓条码扫描与分享工具。专为需要快速、连续扫描并整理条码的场景设计。

## ✨ 主要功能

*   **🚀 快速连续扫描**: 摄像头保持开启状态，实现不间断的条码识别。
*   **✅ 智能去重**: 连续扫描到同一个条码时，列表中只记录一次。
*   **📋 清晰列表**: 所有扫描结果以列表形式清晰展示。
*   **✏️ 灵活管理**:
    *   **单个删除**: 可以轻松删除列表中的任意一个条码。
    *   **一键清空**: 快速清除所有已扫描的记录。
*   **📲 便捷分享**:
    *   **批量复制**: 一键复制所有条码结果到剪贴板。
    *   **文本分享**: 通过系统的分享功能，将所有条码以文本形式发送到其他应用。

## 🛠️ 技术栈

*   **语言**: [Kotlin](https://kotlinlang.org/)
*   **核心框架**: AndroidX (AppCompat, RecyclerView, ConstraintLayout)
*   **相机**: [CameraX](https://developer.android.com/training/camerax)
*   **图像识别**: [Google ML Kit - Barcode Scanning](https://developers.google.com/ml-kit/vision/barcode-scanning)

## 🚀 如何构建

1.  克隆本仓库到您的本地机器。
2.  使用 Android Studio 打开项目。
3.  等待 Gradle 完成项目同步和依赖下载。
4.  直接构建或在您的设备上运行。
